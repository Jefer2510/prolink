Características Principales

Autenticación JWT

Sistema completo de registro, login y autenticación con tokens JWT seguros.

Red Social

Conexiones entre usuarios, posts, likes, comentarios y perfiles profesionales.

Mensajería Real-Time

Sistema de mensajes instantáneos usando WebSocket y STOMP.

API Endpoints Principales

Autenticación

POST /api/v1/auth/login
GET /api/v1/auth/health
Posts y Social
GET /api/v1/posts/feed
POST /api/v1/posts
GET /api/v1/users/profile
